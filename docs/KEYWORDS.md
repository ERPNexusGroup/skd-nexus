# Palabras Clave Estándar

Las siguientes palabras clave están reservadas y categorizadas para su uso en los manifiestos de los componentes.

## Categorías

- **Industria**: `contabilidad`, `hospitalidad`, `educacion`, `salud`
- **Región**: `Ecuador`, `Latam`, `Global`
- **Funcionalidad**: `reportes`, `integracion`, `ui`, `backend`

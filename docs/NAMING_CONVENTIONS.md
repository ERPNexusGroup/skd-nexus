# Convenciones de Nombramiento

## Componentes

- **Módulos**: `mod_<snake_case>` (ej: `mod_hotel_reservations`)
- **Aplicaciones**: `app_<snake_case>` (ej: `app_contracts`)
- **Librerías**: `lib_<snake_case>` o `pkg_<snake_case>` (ej: `lib_pdf_generator`)

## Versiones

Se utiliza [Semantic Versioning 2.0.0](https://semver.org/).
- Formato: `MAJOR.MINOR.PATCH`
